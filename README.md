# ⚔ LifeUp 存档管理器

> 独立桌面应用，直接读写 LifeUp 备份存档（SQLite），无需手机 API。

## 🎯 功能

- 📊 **修行总览** — 金币、等级、任务统计、商品数、成就进度、六维属性
- 📋 **任务管理** — 增删改查，自动去重，完整的字段支持
- 🏪 **商店管理** — 商品列表、编辑、筛选
- 🏆 **成就管理** — 自定义成就 + 系统成就查看
- ⚔ **属性面板** — 技能经验、升级进度
- 🎒 **背包物品** — 库存查看

## 🚀 使用方式

### 桌面版（推荐）

1. 下载 `dist/LifeUp存档管理器.exe`
2. 双击运行
3. 点击「📂 打开存档」选择 LifeUp 备份 ZIP 文件
4. 修改任务/商品/成就
5. 按 `Ctrl+S` 或点「💾 保存」
6. 在手机端恢复存档

### 浏览器版

```bash
pip install flask
python server.py
# 打开 http://localhost:5000
```

## 🏗 技术架构

```
Flask (后端) + pywebview (桌面壳) + PyInstaller (打包)
         ↓
    直接读写 SQLite（存档 ZIP 内）
         ↓
    修改后重新打包 ZIP
```

- **无 API 依赖** — 不连接手机，不调 LifeUp API
- **直接 SQLite 读写** — 修改即时生效
- **自动去重** — 备份恢复产生的重复任务自动合并

## 📁 项目结构

```
lifeup-dashboard/
├── server.py           # Flask 后端（数据库读写、API）
├── index.html          # Web 前端（仪表盘、任务/商品/成就管理）
├── desktop_app.py      # 桌面壳（pywebview 原生窗口）
├── dist/               # 打包好的 EXE
├── 启动.bat            # 浏览器版启动脚本
└── 启动(桌面版).bat    # 桌面版启动脚本
```

## ⚠️ 注意事项

- 修改前请手动备份一份存档
- LifeUp 恢复后可能需要重启 App 才能看到变化
- 系统成就表只读，仅可编辑自定义成就
