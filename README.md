# ⚔ 诸天万界 · LifeUp 存档管理器

> 独立桌面应用，直接读写 LifeUp 备份存档（SQLite），无需手机 API。

## 🎯 功能清单

### 核心数据管理
- 📊 **修行总览** — 金币、等级、任务统计、商品数、成就进度、六维属性
- 📋 **任务管理** — 增删改查、自动去重、重要程度/困难度/技能/奖励全字段
- 🏪 **商店管理** — 商品 CRUD、分类筛选、批量改价
- 🏆 **成就管理** — 自定义成就 + 系统成就查看
- ⚔ **属性面板** — 技能经验、升级进度

### 背包与合成
- 🎒 **背包物品** — 卡片网格/表格双视图、搜索、排序
- 🔮 **合成配方** — 100 配方 CRUD、分类筛选、材料/产出管理
- 🧮 **合成计算器** — 库存匹配、可合成/缺件分析、一键筛选
- 🃏 **卡牌图鉴** — 151 张卡牌查询、稀有度筛选

### 统计与分析
- 📜 **活动记录** — 任务完成/物品变动时间线、日期范围筛选
- 📊 **经济看板** — 11 天收支趋势可视化
- 📈 **成就进度** — 33 个目标实时完成度

### 高级功能
- 🎰 **卡池管理** — 22 卡池、133 效果查询
- 📸 **快照管理** — 数据库快照记录
- 🌌 **位面切换** — 多位面数据管理
- 🔄 **批量操作** — 任务/商品批量启用/禁用/删除/改价
- 📥 **数据导出** — 6 表 JSON + CSV 一键导出
- 🌙 **深色模式** — 暗黑/明亮主题切换

### 可靠性
- 🛡 **WAL 安全** — 保存前强制 checkpoint，杜绝数据丢失
- 💾 **Ctrl+S 保存** — 即时打包回 ZIP，支持手机端恢复

## 🚀 使用方式

### 桌面版（推荐）
1. 下载 `dist/LifeUp存档管理器.exe`（绿色单文件，15.7MB）
2. 双击运行
3. 点击「📂 打开存档」选择 LifeUp 备份 ZIP 文件
4. 修改任务/商品/成就/配方
5. 按 `Ctrl+S` 或点「💾 保存」
6. 在手机端恢复存档

### 浏览器版
```bash
pip install flask
python server.py
# 打开 http://localhost:5000
```

## 🏗 技术架构

```
Flask (后端 API) + 原生 HTML/JS (前端) + pywebview (桌面壳) + PyInstaller (打包)
                        ↓
              直接 SQLite 读写（存档 ZIP → DB → ZIP）
                        ↓
              38+ API 端点 | 110+ 前端函数 | 120KB 单文件
```

- **无 API 依赖** — 不连接手机，不调 LifeUp API
- **直接 SQLite 读写** — 修改即时生效，WAL 安全
- **自动去重** — 备份恢复产生的重复任务自动合并
- **纯 Web 前端** — 无框架依赖，单 HTML 文件
